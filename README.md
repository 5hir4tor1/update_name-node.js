update_name-node.js
===================

Node.jsでUpdate_Name

## Requirement

- twit

## Install
```sh
git clone https://github.com/thr3a/update_name-node.js.git
```
**app.js**の```screen_name```と４つのAPIKeyを記述
```sh
npm install
node app.js
```
